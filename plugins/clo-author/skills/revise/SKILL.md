# Revise

Use this skill for R&R planning and response drafting.

## Read First

- `AGENTS.md`
- `plugins/clo-author/references/codex-workflow.md`
- `plugins/clo-author/references/agent-catalog.md`
- `plugins/clo-author/references/legacy-rules/revision.md`

## Procedure

1. Parse every referee or editor comment.
2. Classify each item as:
   - new analysis
   - clarification
   - rewrite
   - disagreement
   - minor fix
3. Route each item to the appropriate role.
4. Draft a point-by-point response letter and a concrete revision checklist.
5. Flag disagreement items for explicit user approval before drafting pushback.

## Outputs

- Response plan in `quality_reports/`
- Revision checklist in `quality_reports/`
- Draft response letter when requested
